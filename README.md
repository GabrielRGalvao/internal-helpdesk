# Sistema Interno de Tickets (Desafio Técnico)

## Tecnologias Utilizadas
- Laravel 10 / 11
- Vue 3 (Options/Composition API)
- Inertia.js
- Tailwind CSS v4

## Como Rodar o Projeto Localmente

1. Clone o repositório
2. Instale as dependências do PHP: `composer install`
3. Instale as dependências do Node: `npm install`
4. Configure o `.env` e rode as migrations/seeders: `php artisan migrate --seed`
5. Inicie os servidores em terminais separados:
   - `php artisan serve`
   - `npm run dev`
